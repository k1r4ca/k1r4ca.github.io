Hexo建站及主题设置

先来个官方的doc ，多看文档总是好事，其实写的也很清楚了，还有中文

建议通过官方文档了解一下常用的命令 指令 | Hexo

需要安装的环境：Node.js 可以在官网 下载长期维护版LTS

安装完可以在命令行用老规矩确认是否成功（node -v``npm -v）

再就是安装hexo了，利用npm就好

npm install -g hexo-cli +老规矩hexo -v

之后是初始化

在执行完最后一步之后就可以访问localhost:4000来查看

如果像我一样没法成功可能是没有自带主题，自己选个主题照着quick start做应该就行
